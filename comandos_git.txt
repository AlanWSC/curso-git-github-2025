git init .               -> cria um repositórioa git na pasta atual
git status               -> mostra o status do trabalho de cada arquivo
git add nome_arquivo     -> adiiona arquivo para ser commitado
git commit -m "msg"      -> faz commit do que estava para ser commitado
git log                  -> mostra todo o histórico de commit
git diff <nome_arquivo>  -> para saber o que tem de diferença do estado atual para o estado anterior
git reset <nome_arquivo> -> retira um ou mais arquivos que estão pré commitados(estão no add)

## Fluxo de trabalho

1. criar diretório do projeto
2. iniciar o git dentro do diretório criado(git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome_arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
